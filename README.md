# Alan Jiang | Personal Portfolio

An interactive, high-performance portfolio website showcasing projects and technical skills. This site features smooth scrolling, complex GSAP animations, and a 2D physics engine for immersive user interactions.

## 🚀 Live Demo

[alanjiang.xyz](https://www.alanjiang.xyz/)

## ✨ Key Features

- **Immersive Physics:** Interactive 2D physics simulation using Matter.js for dynamic, falling pill-shaped UI elements.
- **Smooth Navigation:** Silky-smooth scrolling experience powered by Lenis, perfectly synchronized with GSAP ScrollTrigger.
- **High-Performance Animations:** Complex scroll-triggered animations, text effects, and transitions built with GSAP.
- **Responsive Design:** Fluid layouts optimized for both desktop and mobile devices using Tailwind CSS.
- **Modern Tech Stack:** Built for speed, scalability, and maintainability.

## 🛠 Tech Stack

| Category       | Technology                               |
| :------------- | :--------------------------------------- |
| **Framework**  | React 18 (Hooks + Functional Components) |
| **Language**   | TypeScript (Strict Mode)                 |
| **Styling**    | Tailwind CSS                             |
| **Animations** | GSAP 3 + @gsap/react                     |
| **Physics**    | Matter.js                                |
| **Scroll**     | Lenis                                    |
| **Build Tool** | Vite                                     |

## 🚀 Getting Started

### Prerequisites

- Node.js (v22+)
- npm

### Local Installation

1. **Clone the repository**

   ```sh
   git clone https://github.com/adj2424/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Start the development server**

   ```sh
   npm run dev
   ```

## 🏗 Scripts

- `npm run dev`: Start the development server.
- `npm run build`: Build the project for production.
- `npm run lint`: Run ESLint.
- `npm run preview`: Preview the production build locally.
